# Glossary of Data Analytics Terms, Terminologies & Acronyms

This glossary provides a concise reference of commonly used terms in data analytics, statistics, and business analysis.

---

## A

**A/B Testing**  
A statistical method of comparing two versions of a variable to determine which performs better.

**API (Application Programming Interface)**  
A set of rules that allows different software systems to communicate and exchange data.

**Aggregation**  
The process of summarizing data (e.g., sum, average, count).

---

## B

**BI (Business Intelligence)**  
Tools and processes used to analyze business data and support decision-making.

**Big Data**  
Extremely large datasets that require specialized tools for storage and processing.

**Box Plot**  
A visualization used to show distribution, median, and outliers.

---

## C

**Categorical Data**  
Data that represents categories or groups rather than numbers.

**Correlation**  
A statistical measure of the relationship between two variables.

**Cohort**  
A group of users or data points sharing a common characteristic over time.

**CSV (Comma-Separated Values)**  
A common file format for storing tabular data.

---

## D

**Dashboard**  
A visual interface displaying key metrics and KPIs.

**Data Cleaning**  
The process of fixing or removing incorrect, missing, or inconsistent data.

**Data Frame**  
A tabular data structure used in Python (Pandas) and R.

**Data Lake**  
A storage system for raw, unprocessed data.

**Data Warehouse**  
A centralized system optimized for structured analytical queries.

---

## E

**EDA (Exploratory Data Analysis)**  
The process of analyzing datasets to summarize their main characteristics.

**ETL (Extract, Transform, Load)**  
A data pipeline process used to move and transform data into a usable format.

---

## F

**Feature Engineering**  
Creating new variables from existing data to improve analysis or modeling.

**Funnel Analysis**  
A method of analyzing user steps in a conversion process.

---

## G

**Group By**  
A SQL and Pandas operation used to aggregate data by categories.

---

## H

**Hadoop**  
A distributed system for processing large datasets.

**Hypothesis Testing**  
A statistical method used to test assumptions about data.

---

## I

**IQR (Interquartile Range)**  
A measure of statistical dispersion used to detect outliers.

**Imputation**  
The process of filling missing values in data.

---

## K

**KPI (Key Performance Indicator)**  
A measurable value used to evaluate success against objectives.

---

## L

**Lagging Indicator**  
A metric that reflects past performance (e.g., revenue).

**Leading Indicator**  
A metric that predicts future performance (e.g., website traffic).

---

## M

**Mean**  
The average of a dataset.

**Median**  
The middle value in a sorted dataset.

**Mode**  
The most frequently occurring value.

**Missing Data**  
Data values that are not recorded or unavailable.

---

## N

**Normalization**  
Scaling data to a standard range (e.g., 0 to 1).

**Null Value**  
A missing or undefined value in a dataset.

---

## O

**Outlier**  
A data point that significantly differs from other observations.

---

## P

**Pandas**  
A Python library for data manipulation and analysis.

**P-value**  
A probability measure used in hypothesis testing.

**Pivot Table**  
A tool used to summarize and reorganize data.

**Population**  
The entire dataset or group being studied.

**Python**  
A programming language widely used in data analytics.

---

## Q

**Quartile**  
Values that divide data into four equal parts.

---

## R

**Regression**  
A statistical method used to model relationships between variables.

**Row**  
A single record in a dataset.

---

## S

**Sample**  
A subset of a population used for analysis.

**SQL (Structured Query Language)**  
A language used to query and manage relational databases.

**Standard Deviation**  
A measure of how spread out data is from the mean.

**Skewness**  
A measure of asymmetry in data distribution.

---

## T

**Time Series**  
Data points collected over time intervals.

**T-Test**  
A statistical test used to compare means between groups.

---

## U

**Univariate Analysis**  
Analysis of a single variable.

---

## V

**Variance**  
A measure of data spread around the mean.

**Visualization**  
Graphical representation of data.

---

## W

**Window Function (SQL)**  
A function that performs calculations across a set of rows related to the current row.

---

## Z

**Z-Score**  
A measure of how many standard deviations a value is from the mean.

---
